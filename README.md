# GCATool
___
A open source too to unpack and repack GCA files.

Be aware that this tool has not been fully tested and may have bugs.
Repacking has not been fully implemented yet, features such as encryption and compression are not supported and may never
be.

___
### Usage
```bash
# Unpacking
GCATool -u file.gca output_directory

# Packing
GCATool -p input_directory output_file.gca
```
